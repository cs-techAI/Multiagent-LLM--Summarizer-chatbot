# Multi-Agent LLM Application

This Streamlit application allows users to summarize text or engage in chat using uploaded documents in PDF or DOCX formats. It utilizes Hugging Face Transformers for summarization and question-answering tasks.

## Features

- **Summarization**: Summarizes text from uploaded documents or JSON input.
- **Chat Agent**: Answers user queries based on the content of uploaded documents.
- **File Upload**: Supports both PDF and DOCX file formats.

## Requirements

- Python 3.7 or higher
- Streamlit
- Transformers
- PyMuPDF
- python-docx
